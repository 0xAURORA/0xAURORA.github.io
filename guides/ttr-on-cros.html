    TTR on ChromeOS

    • Running Toontown Rewritten on Chrome OS

    Thanks to significant improvements to Crostini (a.k.a Linux Beta), it is now easier than ever to run Toontown Rewritten on a Chromebook! Here I'll discuss using Crostini and Flatpak to run the game, but you should check out crouton and GalliumOS if that doesn't do the trick for you (I may update this page with more info on those later!). Note that none of these methods work on non-x86_64 chromebooks. Until emulation improves on ARM platforms, you're most likely out of luck if you've got an ARM chromebook.

    Using Crostini

    To install Toontown Rewritten using Crostini, you'll first want to install Linux (Beta). You can install Linux (Beta) by opening up your Chromebook's system settings, finding Linux (Beta) on the side, and clicking "Install"

    If the installation fails, reboot your device and try again. Otherwise, open up the linux terminal by clicking the little dot icon in the bottom left corner to open up the app launcher and launching the Terminal app under the Linux section.

    Once you've got the terminal open, we'll next need to make sure everything is going to work. First you should check that your process type is compatible with Toontown Rewritten. Type the following command into the terminal as is followed by a press of the enter key.

    uname -m

    You should get some output a little bit like this.

    If you don't see "x86_64," then, unfortunately, your device will not work with the game. Otherwise, you should next check to make sure GPU acceleration is enabled. Install mesa-utils using the following command.

    sudo apt update && sudo apt install -y mesa-utils

    After you've got mesa-utils successfully installed, run the following command to verify GPU acceleration is enabled.

    glxinfo -B

    After you've run glxinfo, you should see some output similar to the above. The important bit is the "direct rendering: Yes" (outlined above). If it says no, then you will need to enable GPU acceleration. You can do so by opening a chrome browser, going to chrome://flags, searching for "Crostini GPU Support" and enabling it. Once you've done that, restart your device for good measure and try running glxinfo -B again.

    After you've successfully verified that hardware acceleration is enabled, we're almost ready to install the game. Before doing that, we'll need to get Flatpak using the following command.

    sudo apt install -y flatpak

    After Flatpak is installed, install the game like so:

    flatpak --user install https://www.xytime.xyz/flatpaks/ttr/toontown.flatpakref

    Since you're installing a flatpak for the first time, you'll probably be prompted to add flathub as a remote. Make sure you say yes ('y') to this or the flatpak won't install. I was not in my screenshot because I've used flatpak before.

    Press 'y' again to confirm the changes and wait a bit while it downloads Toontown Rewritten and the libraries required to run it. You'll probably have a few more things to download than me as I've already used flatpak before.

    After installation is complete, you should see Toontown Rewritten appear in the Linux section of your App Launcher (next to the terminal app you launched earlier). If you don't, try restarting as sometimes it's needed to get it to show up. If you still don't see it, you can manually run the flatpak using the following command.

    flatpak run xyz.xytime.Toontown

    Once starting the App, you should see the Toontown Rewritten launcher appear and you'll be able to login as normal. It might not work perfectly as Chromebooks are a somewhat diverse group of devices and Crostini is still in beta (hence, Linux Beta). If this doesn't work for you or you experience issues, try Crouton or GalliumOS (as linked above).
